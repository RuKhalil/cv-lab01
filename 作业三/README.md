# 实验3 DFT与DCT对比实验

## 实验目的
1. 掌握一维DFT与DCT-II的数学定义与基函数含义。
2. 理解DFT与DCT的输出系数差异、频域分解思想的共同点。
3. 对比周期延拓（DFT）与镜像延拓（DCT）的边界特性。
4. 通过频谱图与能量集中性分析，理解DCT在图像处理中的优势。

## 实验原理
1. DFT：基函数为复指数函数，输出复数系数，隐含周期延拓。
2. DCT-II：基函数为实余弦函数，输出实数系数，隐含镜像延拓。
3. 两者均为正交变换，可将空域信号转换到频域，DCT能量集中性更优。

## 实验步骤
1. 读取灰度图像。
2. 实现周期延拓与镜像延拓，对比边界效果。
3. 计算图像的DFT与DCT变换，生成频谱图。
4. 分析能量集中性，对比两种变换的差异。
5. 保存结果图并完成理论分析。

## 实验结果与分析
1. 边界延拓：
   - DFT周期延拓：边界出现明显跳变，产生虚假高频。
   - DCT镜像延拓：边界连续平滑，无跳变失真，更符合图像特性。
2. 频谱对比：
   - DFT频谱能量分布较分散，高频分量多。
   - DCT频谱能量高度集中在低频区域，压缩性能更强。
3. 系数差异：
   - DFT输出复数（幅度+相位）。
   - DCT输出实数，计算更简单高效。
4. 共同点：均为正交变换，实现空域-频域转换，具备能量集中特性。

## 实验结论
1. DFT基函数为复指数，输出复数；DCT基函数为余弦函数，输出实数。
2. DCT采用镜像延拓，边界连续无跳变，优于DFT的周期延拓。
3. DCT能量集中性更强，是图像压缩（如JPEG）的核心技术。
4. DCT在计算效率、边界友好性、压缩性能上均优于DFT。
