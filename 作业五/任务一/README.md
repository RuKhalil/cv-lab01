# 作业五 任务一：相似、仿射与透视变换实验

## 实验目的
1.  理解相似变换、仿射变换与透视变换的数学模型与特性。
2.  对比三种变换对直线、平行线、垂直线和圆的几何性质的影响。
3.  掌握 OpenCV 中 `cv2.warpAffine` 与 `cv2.warpPerspective` 的使用方法。

## 项目文件

├── geo_transform.py         # 三种变换的实现代码
├── geo_transform_result.png # 三种变换对比结果图

## 实验原理
1.  **相似变换**：包含平移、旋转和等比例缩放，保持直线、平行、垂直和圆的形状不变。
2.  **仿射变换**：包含平移、旋转、缩放和剪切，保持直线和平行性，但不保持垂直和圆的形状。
3.  **透视变换**：模拟相机的投影变换，仅保持直线，不保持平行、垂直和圆的形状。

## 实验步骤
1.  准备测试图，绘制矩形、圆、平行线和垂直线。
2.  对图像分别施加相似变换、仿射变换和透视变换。
3.  观察并记录每种变换对几何性质的影响。
4.  绘制三种变换的对比图，保存结果。

## 实验结果与分析
1.  **相似变换**：所有几何性质均保持不变，仅整体发生旋转、缩放和位置变化。
2.  **仿射变换**：直线和平行线保持不变，垂直线不再垂直，圆被拉伸为椭圆。
3.  **透视变换**：仅直线保持不变，平行线、垂直线和圆的形状均发生畸变。

## 实验结论
1.  相似变换保距保角，仿射变换保平行不保角，透视变换仅保共线性。
2.  三种变换的差异源于它们的自由度不同，自由度越高，能模拟的畸变越强，但保留的几何性质越少。
