# 作业五 任务二：透视畸变校正实验

## 实验目的
1.  理解透视变换在图像校正中的应用原理。
2.  掌握使用 `cv2.getPerspectiveTransform` 和 `cv2.warpPerspective` 校正透视畸变的方法。
3.  实现对带有透视畸变的平面文档图像的校正，使其变为正视视角。

## 项目文件
├── paper.jpg               # 带有透视畸变的A4纸图像
├── perspective_correction.py # 透视校正实现代码
├── perspective_correction.png # 校正前后对比结果图
└── README.md               # 实验说明文档

## 环境依赖
Python 3.x
opencv-python
numpy
matplotlib

## 安装依赖
pip install opencv-python numpy matplotlib

## 运行方法
将待校正图像命名为 `paper.jpg`，与代码放在同一目录下，执行：
python perspective_correction.py

## 实验原理
透视校正通过以下步骤实现：
1.  手动选取畸变图像中A4纸的四个角点。
2.  定义校正后标准A4纸的四个目标点（保持1:1.414的宽高比）。
3.  计算透视变换矩阵，将畸变图像映射为正视视角的矩形图像。

## 实验步骤
1.  读取带有透视畸变的A4纸图像。
2.  选取A4纸的四个角点（左上、右上、左下、右下）。
3.  定义标准A4纸的目标尺寸和坐标。
4.  计算透视变换矩阵并应用变换，得到校正后的图像。
5.  对比校正前后的效果，保存结果。

## 实验结果与分析
校正前的图像存在明显的近大远小透视畸变，文字和线条倾斜。
校正后的图像变为正视视角，文字横平竖直，符合标准A4纸的比例，畸变被有效消除。

## 实验结论
1.  透视变换可以有效校正平面图像的透视畸变，将斜拍的文档转为正视视角。
2.  校正效果的关键在于准确选取四个角点，以及保持目标图像的宽高比。
